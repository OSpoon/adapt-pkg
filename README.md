# init-pkg

Initialize the library name, description, repository, readme, and license for MIT

## Usage

```shell
npx init-pkg -n test-lib -d "一个测试库" -a  OSpoon -e "zxin088@gmail.com"
```

```shell
npx init-pkg --boot
```

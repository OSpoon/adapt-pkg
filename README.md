# adapt-pkg

Initialize the library name, description, Github, Readme, and MIT License

## Usage

```bash
npx adapt-pkg -n test-lib -d "一个测试库" -a  OSpoon -e "zxin088@gmail.com"
```

```bash
npx adapt-pkg --boot
```
